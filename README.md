# SoundInbox

Multi-account audio inbox for macOS — manage notifications from all your sound-enabled accounts in one place.

**Free forever** with optional Pro upgrade ($9.99) for multi-account support.

## Download

Coming soon — watch this repo for the first release.

## Features

- Multi-account audio inbox
- Unified notification management
- Menu bar integration
- Sparkle auto-updates

### Free

- 1 account
- Full app functionality

### Pro ($9.99 one-time)

- Up to 5 accounts
- Priority support

## Install

### Direct Download

Download the latest DMG from [GitHub Releases](https://github.com/drolosoft/sound-inbox-mac/releases).

### Homebrew

```bash
brew install --cask drolosoft/tap/soundinbox
```

## Requirements

- macOS 13.0 (Ventura) or later
- Apple Silicon or Intel
